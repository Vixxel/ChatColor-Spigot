# ChatColor

A modern GUI-based chat color plugin for Spigot 1.21.11.

## Features

- 🎨 GUI-based color selection
- 🔒 Permission-based chat colors
- 💬 Colored chat messages
- 💾 Persistent player data
- ⚡ Lightweight
- 📦 Built with Java 21

## Commands

| Command | Description |
|---------|-------------|
| `/chatcolor` | Displays help |
| `/chatcolor gui` | Opens the chat color menu |

## Permissions

```
chatcolor.*
chatcolor.black
chatcolor.darkblue
chatcolor.darkgreen
chatcolor.darkaqua
chatcolor.darkred
chatcolor.darkpurple
chatcolor.gold
chatcolor.gray
chatcolor.darkgray
chatcolor.blue
chatcolor.green
chatcolor.aqua
chatcolor.red
chatcolor.pink
chatcolor.yellow
chatcolor.white
```

## Requirements

- Java 21
- Spigot 1.21.11

## Installation

1. Download the latest release.
2. Place the JAR into your server's `plugins` folder.
3. Restart the server.
4. Configure permissions.
5. Enjoy!
