# 🎨 ChatColor

A lightweight, GUI-based chat customization plugin for **Spigot 1.21.x**.

ChatColor allows players to customize their chat messages using colors and formatting through an intuitive inventory GUI. Permissions are fully supported, and player selections are automatically saved across server restarts.

---

## ✨ Features

- 🎨 16 Chat Colors
- ✨ Chat Formatting
  - Bold
  - Italic
  - Underline
  - Strikethrough
  - Magic (Obfuscated)
- 🖥️ Simple GUI Interface
- 🔒 Permission-Based System
- 💾 Persistent Player Data
- 🚫 Reset Chat Color & Formatting
- ⚡ Lightweight
- ☕ Java 21
- 📦 Spigot 1.21.x Support

## 📥 Installation

1. Download the latest release.
2. Place the jar into your server's `plugins` folder.
3. Start or restart your server.
4. Configure permissions.
5. Players can now use `/chatcolor gui`.

---

## 💬 Commands

| Command | Description |
|----------|-------------|
| `/chatcolor` | Displays plugin usage. |
| `/chatcolor gui` | Opens the Chat Color GUI. |

## 🔑 Permissions

### Chat Colors

```

chatcolor.darkblue
chatcolor.darkgreen
chatcolor.darkaqua
chatcolor.darkred
chatcolor.darkpurple
chatcolor.gold
chatcolor.gray
chatcolor.darkgray
chatcolor.blue
chatcolor.green
chatcolor.aqua
chatcolor.red
chatcolor.pink
chatcolor.yellow
chatcolor.white
```

### Chat Formatting

```
chatformat.bold
chatformat.italic
chatformat.underline
chatformat.strikethrough
chatformat.magic
```

### Grant Everything

```
chatcolor.*
chatformat.*
```

## 📁 Configuration

The plugin automatically generates:

```
plugins/
└── ChatColor/
    ├── config.yml
    ├── messages.yml
    └── players.yml
```

Player selections are automatically saved and restored.
